Guía de registro Universitarios

Bienvenido a la guía informativa para estudiantes universitarios. Es importante seguir correctamente cada paso del proceso.

La fecha límite de inscripción es obligatoria. No cumplir con este requisito puede afectar tu inscripción.

Pasos para la inscripción

  1. Revisar materias disponibles.
  2. Realizar el pago correspondiente.
  3. Registrar materias en el sistema.
  4. Confirmar horario final.

Recomendaciones generales

  • Consultar avisos oficiales.
  • Guardar comprobantes.
  • Organizar tu tiempo.
  • Buscar apoyo académico si es necesario.
  • Tener el tiempo suficiente para cada materia.
