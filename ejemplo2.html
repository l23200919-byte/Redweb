Acerca del sistema

Este sistema permite reportar problemas técnicos dentro de la empresa. Los usuarios pueden registrar fallas para que el área de soporte las atienda.

Servicios disponibles

  • Reporte de fallas
  • Seguimiento de tickets
  • Soporte técnico

Tipos de problemas

  • Internet lento
  • Equipo no enciende
  • Error en software
  • Impresora sin funcionar

El sistema organiza los reportes por prioridad y estado.
